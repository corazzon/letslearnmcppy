# FastMCP Basic Server Example

FastMCP 프레임워크를 사용한 기초 MCP 서버 구현 예제

## 🚀 Quick Start

```bash
# 의존성 설치
uv add fastmcp

# 서버 실행
python examples/fastmcp_basic/server.py
```
